Interactive Sales Dashboard Preview

This image provides a static preview of the interactive sales dashboard. To experience the full interactivity, please follow the setup instructions below.

How to run the interactive dashboard:

  1. Ensure you have Python 3.8+ and pip installed.
  2. Clone or download the project repository.
  3. Open your terminal/command prompt in the project's root directory.
  4. Install dependencies: pip install -r requirements.txt
  5. Generate data: python data_generator.py
  6. Run the Dash app: python app.py
  7. Open the URL (usually http://127.0.0.1:8050/) provided in your terminal in a web browser to view the interactive dashboard.
